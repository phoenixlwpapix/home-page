# STUDIO YYH. - Personal Portfolio

一个现代化的个人作品集网站，展示我的项目和技能。

## 关于我

**Youthful, Yummy, Happy** - YYH 的个人主页

热爱跆拳道、网球、游泳和计算机编程。通过技术创造有趣的应用，让生活更美好。

## 技术栈

- **React 19** - 现代化的用户界面库
- **Vite** - 快速的构建工具和开发服务器
- **Tailwind CSS** - 实用的 CSS 框架
- **Lucide React** - 精美的图标库
- **ESLint** - 代码质量检查

## 功能特性

- 🌙 深色/浅色主题切换
- 📱 响应式设计，适配各种设备
- 🚀 快速加载和流畅的动画效果
- 🎨 现代化的 UI 设计
- 🔗 社交媒体链接集成

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```
